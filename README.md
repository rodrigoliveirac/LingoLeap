# LingoLeap App 🍎
That's the new way to learn english! :)

## Features 🔬

🔎 Word search.
> 
📖 Exploration of all parts of speech for the searched word, such as nouns, verbs, pronouns, and more.
> 
🌍 Translation into 111 idioms.
> 
🗣️ See how to pronounce the English word.
>
